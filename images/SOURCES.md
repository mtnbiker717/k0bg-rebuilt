## License Requirements

- **Public Domain / CC0**: No attribution required, but credited here for transparency
- **CC-BY**: Must credit author name and link to original
- **CC-BY-SA**: Must credit author, link to original, and note any modifications. Derivative works must use same license.
- **CC-NC / CC-ND**: NOT used on this site

## Adding New Images

1. Confirm the license allows non-commercial use and redistribution
2. Download and save to the appropriate `images/` subdirectory
3. Add an entry to this file with filename, source URL, author, and license
4. Add `<figure>` markup in the relevant article HTML
5. Optimize: max 1200px wide, compress JPEGs to ~80% quality
